# 3D Street Sprint

A browser-based 3D car racing game where you can:

- Customize your car color in the garage.
- Race against bot drivers with Easy, Medium, and Hard levels.
- Earn coins based on your race result and distance.
- Buy one-use powerups (Nitro Burst, Turbo Tires, Coin Magnet) from the shop.
- Tune settings with poly mode (high/low), graphics quality (high/low), and selectable resolution from 800×600 up to max screen size.

## Run locally

Because the project uses ES modules, run it through a local web server:

```bash
python3 -m http.server 8080
```

Then open `http://localhost:8080`.
